HTML Tags and Descriptions

Tag Description
<html> Declares the Web page to be written in HTML
<head> Delimits the page's head
<title> Defines the title (not displayed on the page)
<body> Delimits the page's body
<b> Set in boldface
<i> Set in italics
<center> Center... on the page horizontally
<ul> Brackets an unordered (bulleted) list
<ol> Brackets a numbered list
<li> Brackets an item in an ordered or numbered list
